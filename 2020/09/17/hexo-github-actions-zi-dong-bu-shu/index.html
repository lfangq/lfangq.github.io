4.错误问题

1.问题：引用的样式没有自动部署

原因：git clone themes/<主题名>下载的目录中有隐藏的.git文件，导致git push时该目录下的文件不能添加

解决方法：删除.git文件

2.问题：git@github.com: Permission denied (publickey)
原因：github权限问题
解决方法：检查github密钥是否正确
